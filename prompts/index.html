Yingzao Fashi
3D Architecture Visualization · Single HTML · Three.js
Kimi K2.6 → Opus 4.8 → GPT 5.5 →

Generate a "Yingzao Fashi" (Building Standards) disassembly animation based on a photo of an ancient building. The user uploads a photo of a Chinese ancient building (such as the Yingxian Wooden Pagoda or the Main Hall of Foguang Temple). The model needs to:

  • Identify the building type and major structural components (dougong/bracket sets, columns, beams, fang/tie-beams, purlins, rafters)
  • Generate a simple Three.js 3D model, demonstrating the assembly process in the actual traditional construction order (first erect the columns, then place the beams, then install the dougong brackets, and finally put on the roof)
  • Each component should display its name and function as a floating/hover label
  • Be able to use a slider to control "age erosion," observing the process of wood weathering and paint peeling off
  • Support toggling a "structural X-ray/transparent" mode, hiding the exterior walls to view the internal mortise-and-tenon joints

Output: a single HTML file.
